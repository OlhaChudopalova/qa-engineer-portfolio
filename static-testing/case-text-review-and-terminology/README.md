# Case: Text Review and Terminology

This case demonstrates static testing focused on text review, terminology consistency, and content clarity across the product.

The work includes:
- Review of system and notification texts
- Identification of inconsistencies in terminology and naming
- Creation of a unified terminology table
- Example of content review for the Home page

The goal was to improve text quality, reduce ambiguity, and ensure consistent communication with users through clear and unified language.

**Files included in this case:**
- `notification-text-review.md` — analysis of system and notification texts
- `terminology-unification-table.md` — unified terminology table used across the project
- `home-page-text-review.md` — example of Home page content review

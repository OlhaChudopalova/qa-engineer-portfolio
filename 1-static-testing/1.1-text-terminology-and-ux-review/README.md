# 1.1 – Text, Terminology, and UX Review

This section contains static testing focused on reviewing notification texts, unifying terminology across the project, and analyzing Home Page texts from a UX perspective.

The objective of this review was to:
- improve text clarity and readability,
- eliminate ambiguity,
- ensure consistent terminology,
- enhance overall user communication and UX quality.

---

## Included Cases

### **1.1.1 – Notification Text Review**
Static testing of notification and email texts, including:
- terminology consistency,
- grammar and wording issues,
- UX clarity,
- security-related considerations.

---

### **1.1.2 – Home Page Text and UX Review**
Comprehensive review of Home Page texts with a focus on:
- clarity and consistency,
- UX alignment,
- visual-text interaction,
- interaction expectations (clickability, hover behavior, etc.).

The findings are divided into:
- **Quick Fixes** (minor improvements with immediate impact),
- **Long-term UX Improvements** (structural and interaction-related recommendations).

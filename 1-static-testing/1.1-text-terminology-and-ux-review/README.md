# 1.1 – Text, Terminology, and UX Review
This section contains static testing focused on reviewing notification texts, unifying terminology across the project, and analyzing Home Page texts from a UX perspective.  
The goal of this review was to improve text quality, reduce ambiguity, and ensure consistent and clear communication with users.
## Included cases:
- **1.2.1 – Notification Text Review**  
  Static testing of notification and email texts, including terminology consistency, grammar issues, UX clarity, and security-related considerations.  

- **1.2.2 – Terminology Unification Table**  
  A consolidated table unifying terminology used across the project to ensure consistency.  

- **1.2.3 – Home Page Text and UX Review**  
  Review of Home Page texts with a focus on clarity, UX, and consistency. Some findings were identified as long-term UX improvements.  
Screenshots and visual references are stored in the `home-page-assets` folder.
